Introduction

Using the instructions on https://sr320.github.io/course-fish546-2025/assignments/01-blast.html, this document will show the different components required to complete a BLAST protein analysis. There are X sections, with the final product displaying tables of the results.

Links used: NCBI software: https://ftp.ncbi.nlm.nih.gov/blast/executables/blast+/LATEST/ Swiss Prot protein sequences .gz file: https://ftp.uniprot.org/pub/databases/uniprot/current_release/knowledgebase/complete/uniprot_sprot.fasta.gz *Query sequence FASTA file: https://eagle.fish.washington.edu/cnidarian/Ab_4denovo_CLC6_a.fa

4.RUNNING BLAST

Translates nucleotide sequences in Ab_4denovo_CLC6_a.fa file in all six reading frames. Comparing those translated proteins against the Swiss-Prot section of UniProt (uniprot_sprot_r2023_01). Outputs top hits (1 per query) with very stringent matches (evalue 1E-20). Produces output in tabular format saved to: output/Ab_4-uniprot_blastx.tab.

/home/jovyan/applications/ncbi-blast-2.16.0+/bin/blastx \
-query /home/jovyan/applications/data/Ab_4denovo_CLC6_a.fa \
-db ../../blastdb/uniprot_sprot_r2023_01 \
-out /home/jovyan/applications/output/Ab_4-uniprot_blastx.tab \
-evalue 1E-20 \
-num_threads 1 \
-max_target_seqs 1 \
-outfmt 6
